# Tic Tac Toe

A simple Tic Tac Toe game for two players, built with React and TypeScript.

## Live Demo

You can play the game directly online at [Vercel Deployment](ta-te-ti-liart.vercel.app/). No installation required!

## Features

- Two-player game (X and O) on a 3x3 board.
- Alternating turns with current player indicator.
- Automatic winner and draw detection.
- Cumulative scores.
- Buttons to reset the game or scores.
- Responsive and accessible design.
