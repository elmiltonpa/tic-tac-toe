# Tic Tac Toe

A simple Tic Tac Toe game for two players, built with React and TypeScript.

## Live Demo

You can play the game directly online at <a href="https://ta-te-ti-liart.vercel.app/" target="_blank">Vercel Deployment</a>. No installation required!

## Features

- Two-player game (X and O) on a 3x3 board.
- Alternating turns with current player indicator.
- Automatic winner and draw detection.
- Cumulative scores.
- Buttons to reset the game or scores.
- Responsive and accessible design.

## How to Run It Locally

1. Install dependencies: `pnpm install`
2. Run the dev server: `pnpm run dev`
